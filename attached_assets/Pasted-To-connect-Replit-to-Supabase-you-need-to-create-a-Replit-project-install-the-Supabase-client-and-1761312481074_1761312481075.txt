To connect Replit to Supabase, you need to create a Replit project, install the Supabase client, and securely store your credentials. Here’s a step-by-step guide:

Step 1: Create a Supabase Project
Go to the Supabase dashboard.

Create a new project and note down:

Project URL (found under Settings > API → “Project URL”)

Anon (public) key (under Settings > API → “anon public”)

These are necessary for client access from Replit.​

Step 2: Create a Replit Project
Visit Replit.com and click “Create Repl”.

Choose Node.js or Python, depending on your language.

Open the Repl workspace.

Step 3: Install Supabase Client Library
For JavaScript/Node.js:

text
npm install @supabase/supabase-js
For Python:

text
pip install supabase
This installs the SDK so you can query your Supabase database.​

Step 4: Add Environment Variables
In Replit, open the Secrets tab (the lock icon in the left sidebar).

Add the following environment variables:

SUPABASE_URL → your project URL

SUPABASE_KEY → your anon/public key
This ensures your credentials are not exposed in your source code.​

Step 5: Initialize Supabase in Code
JavaScript/Node.js Example:

javascript
import { createClient } from "@supabase/supabase-js";

const supabaseUrl = process.env.SUPABASE_URL;
const supabaseKey = process.env.SUPABASE_KEY;
const supabase = createClient(supabaseUrl, supabaseKey);

// Example query
const { data, error } = await supabase.from('messages').select('*');
console.log(data, error);
Python Example:

python
from supabase import create_client
import os

url = os.getenv("SUPABASE_URL")
key = os.getenv("SUPABASE_KEY")
supabase = create_client(url, key)

data = supabase.table("messages").select("*").execute()
print(data)
These examples show how to initialize the Supabase client and interact with the database directly from Replit.​

Step 6: Test the Connection
Run the Repl.

Check the console for output from your test query.

You are now connected to your Supabase backend.

This setup works for authentication, real-time features, and Postgres queries, enabling you to create full-stack applications using Replit and Supabase seamlessly
